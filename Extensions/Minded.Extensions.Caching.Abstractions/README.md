# Minded.Extensions.Caching.Abstractions

Caching abstractions including cache key generation interfaces and attributes.

## Features

- IGenerateCacheKey interface
- Cache attribute definitions
- Cache key generation contracts

## Installation

```bash
dotnet add package Minded.Extensions.Caching.Abstractions
```
