# Minded.Extensions.OData

Core OData utilities and extensions for Minded Framework, providing foundational OData support.

## Features

- OData query parsing utilities
- OData expression builders
- Integration helpers for ASP.NET Core OData
- Query composition support

## Installation

```bash
dotnet add package Minded.Extensions.OData
```

## Usage

See the [main documentation](https://github.com/norcino/Minded) for comprehensive usage examples.
