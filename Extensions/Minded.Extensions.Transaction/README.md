# Minded.Extensions.Transaction

Transaction decorator for automatic database transaction management with support for nested transactions and configurable isolation levels.

## Features

- Automatic transaction scope management
- Nested transaction support
- Configurable isolation levels
- Automatic rollback on failure
- Timeout configuration

## Installation

```bash
dotnet add package Minded.Extensions.Transaction
```
